CausalELM.ActivationFunctions.binary_step — Function
binary_step(x)

Apply the binary step activation function to a real number.

Examples

julia> binary_step(1)
1

Apply the binary step activation function to an array.

CausalELM.ActivationFunctions.softmax — Function
